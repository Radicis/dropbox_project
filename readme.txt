1. Log in to the application via alloydbox.appspot.com using Google openID
2. On the main page the files uploaded by the logged in user are displayed
3. To upload a file, click on the browse button below, locate the file you wish to upload and click the upload button
4. To delete or download a file you have uploaded, check the checkbox beside the file (only 1 at a time or an error is generated) and then click the relevant button Delete/Download.
